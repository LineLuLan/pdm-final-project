// Hướng dẫn cấu hình Lombok cho VS Code
// 1. Mở VS Code, nhấn Ctrl+Shift+X để mở Extensions.
// 2. Tìm "Lombok Annotations Support for VS Code" (tác giả GabrielBB) và cài đặt.
// 3. Nếu vẫn lỗi, thêm vào settings.json:
//    "java.configuration.updateBuildConfiguration": "automatic",
//    "java.import.maven.enabled": true
// 4. Sau khi cài đặt xong, hãy chạy lại lệnh:
//    mvn clean install
// 5. Đóng/mở lại VS Code nếu cần.
